# HABIT-FLOW

HabitFlow es una aplicación web para seguimiento de hábitos personales.

## Funcionalidades principales

- Landing page del proyecto.
- Registro, inicio y cierre de sesión.
- Manejo de sesión activa con localStorage.
- Perfil de usuario y preferencias.
- Dashboard principal.
- Administración de hábitos.
- Estadísticas y visualización de progreso.

## Tecnologías

- React
- Vite
- HTML
- CSS
- JavaScript
- JSON
- localStorage

## Cómo ejecutar

```bash
npm install
npm run dev
```

Luego abrir la URL que muestre Vite en la terminal.

## División de funcionalidades

### Marcelo Loayza

- El usuario podrá registrarse, iniciar y cerrar sesión en la plataforma mediante validación de credenciales y manejo de sesión activa.
- El usuario podrá visualizar y editar su información personal y preferencias dentro de la aplicación.

### Fabrizio Zorrilla

- El usuario podrá visualizar el dashboard principal con hábitos del día, progreso semanal, rachas y métricas generales.
- El usuario podrá marcar y desmarcar hábitos como completados, actualizando el progreso diario del usuario.

### Santiago Franco Baanante

- El usuario podrá administrar hábitos, permitiendo crearlos, editarlos, eliminarlos y visualizarlos.
- El usuario podrá visualizar hábitos organizados mediante listas, categorías y diferentes secciones dentro de la plataforma.

### Alejandro Antonio Ortiz Delgado

- El usuario podrá visualizar estadísticas relacionadas a hábitos completados, rendimiento y actividad reciente.
- El usuario podrá visualizar un diagrama de progreso, top 5 de mejores hábitos y el mejor hábito que cumple identificado con la cantidad de días de racha.
